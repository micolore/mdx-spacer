# MDX Line Spacer

这是一个专为 MDX 开发者设计的实用工具，能够一键为文档每一行末尾添加两个空格，确保 Markdown 在渲染时能够正确换行。

## 功能特性 ✨

- **右键菜单集成**：在 `.mdx` 文件中点击右键即可快速处理。
- **智能过滤**：自动跳过空行，避免产生多余的空格。
- **命令面板支持**：支持通过 `Ctrl+Shift+P` 搜索命令触发。

## 使用方法 🚀

1.  打开任意 `.mdx` 文件。
2.  **方式 A**：在编辑器代码区域点击**鼠标右键**，选择 `MDX: 每行末尾加双空格`。
3.  **方式 B**：按下 `Ctrl + Shift + P`，输入 `MDX` 并选择对应命令。

## 安装方式 🛠️

1.  从 Release 页面下载 `.vsix` 文件。
2.  在 VS Code 插件面板点击右上角的三个点 `...`。
3.  选择 **Install from VSIX...** 并选中下载好的文件。

## 开发初衷

在 MDX 中，标准的 Markdown 换行通常需要行尾有两个空格。手动输入非常麻烦，本插件旨在自动化这一枯燥的过程。
